# Frontend Visual Refresh Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Refresh every TestBoard frontend route into a coherent professional-blue desktop operations console without changing application behavior or data contracts.

**Architecture:** Add three display-only shared presentation components and semantic theme tokens, then migrate the shell and each feature surface onto those primitives. Route pages keep ownership of query and filter state; feature components keep business-specific tables and charts; API clients, hooks, routes, and backend code remain unchanged.

**Tech Stack:** React 19.2.7, TypeScript 6.0, Vite 8.1, Tailwind CSS 4.3, shadcn/ui 4.12, Lucide React 1.22, Recharts 3.3, Vitest 4.1, Testing Library 16.3, pnpm 10.34.

## Global Constraints

- Work only under `frontend/` except for this implementation-plan document.
- Support desktop viewports starting at 1280 CSS pixels; use a 224-pixel sticky sidebar, 32-pixel main padding, and a 1000-pixel minimum content width.
- Do not add mobile navigation, mobile table transformations, a dark-theme toggle, new dependencies, backend changes, or new product capabilities.
- Preserve routes, API clients, React Query hooks, query parameters, pagination, retries, empty detection, report URLs, and pass-rate semantics.
- Preserve existing business copy, headings, filter labels, result labels, and accessible table names unless this plan names the exact replacement.
- Prefer shadcn/ui primitives and Lucide icons; keep `src/components/ui/` as the base-control layer.
- Put cross-feature display components in `src/components/`; keep business-specific presentation in `src/features/<feature>/`; add no barrel exports.
- Every behavior-facing markup change starts with a focused failing test.
- Run commands from `frontend/` unless a step explicitly says otherwise.

import { useId, type ReactNode } from "react";

import { cn } from "@/lib/utils";

export type PageHeaderProps = {
  eyebrow?: ReactNode;
  title: ReactNode;
  description: string;
  status?: ReactNode;
  meta?: ReactNode;
  action?: ReactNode;
  className?: string;
};

export function PageHeader({
  eyebrow,
  title,
  description,
  status,
  meta,
  action,
  className,
}: PageHeaderProps) {
  const titleId = useId();

  return (
    <header
      data-slot="page-header"
      aria-labelledby={titleId}
      className={cn("flex items-start justify-between gap-6", className)}
    >
      <div className="flex min-w-0 flex-col gap-2">
        {eyebrow ? (
          <div className="text-primary text-xs font-semibold tracking-[0.14em] uppercase">
            {eyebrow}
          </div>
        ) : null}
        <div className="flex flex-wrap items-center gap-3">
          <h2 id={titleId} className="text-2xl font-semibold tracking-tight">
            {title}
          </h2>
          {status}
        </div>
        <p className="text-muted-foreground max-w-3xl text-sm">{description}</p>
        {meta ? (
          <div className="text-muted-foreground text-xs">{meta}</div>
        ) : null}
      </div>
      {action ? <div className="shrink-0">{action}</div> : null}
    </header>
  );
}

import type { ReactNode } from "react";
import type { LucideIcon } from "lucide-react";

import { cn } from "@/lib/utils";

export type MetricTone = "default" | "success" | "warning" | "destructive";

export type MetricCardProps = {
  label: string;
  value: ReactNode;
  description?: ReactNode;
  icon: LucideIcon;
  tone?: MetricTone;
  action?: ReactNode;
};

const toneClasses: Record<MetricTone, string> = {
  default: "border-l-primary",
  success: "border-l-success",
  warning: "border-l-warning",
  destructive: "border-l-destructive",
};

const iconClasses: Record<MetricTone, string> = {
  default: "bg-primary/10 text-primary",
  success: "bg-success/10 text-success",
  warning: "bg-warning/15 text-warning-foreground",
  destructive: "bg-destructive/10 text-destructive",
};

export function MetricCard({
  label,
  value,
  description,
  icon: Icon,
  tone = "default",
  action,
}: MetricCardProps) {
  return (
    <Card
      role="article"
      aria-label={label}
      className={cn("border-l-4", toneClasses[tone])}
    >
      <CardContent className="flex min-h-32 flex-col justify-between gap-3 p-4">
        <div className="flex items-start justify-between gap-3">
          <div>
            <p className="text-muted-foreground text-sm font-medium">{label}</p>
            <p className="mt-2 text-3xl font-semibold tracking-tight tabular-nums">
              {value}
            </p>
          </div>
          <div
            className={cn(
              "flex size-9 items-center justify-center rounded-lg",
              iconClasses[tone],
            )}
          >

}

import { useId, type ReactNode } from "react";

import {
  Card,
  CardContent,
  CardDescription,
  CardHeader,
  CardTitle,
} from "@/components/ui/card";
import { cn } from "@/lib/utils";

export type DataPanelProps = {

  description?: string;
  meta?: ReactNode;
  action?: ReactNode;
  children: ReactNode;
  className?: string;
  contentClassName?: string;

export function DataPanel({

  description,
  meta,
  action,
  children,
  className,
  contentClassName,
}: DataPanelProps) {

  return (
    <Card

      aria-labelledby={titleId}
      className={cn("overflow-hidden", className)}

          </div>
          {action ??

              <div className="text-muted-foreground text-sm">{meta}</div>
            ) : null)}
        </div>
      </CardHeader>
      <CardContent className={cn("p-5", contentClassName)}>
        {children}
      </CardContent>
    </Card>
  );
}
